Install Local Helper

Detecting local Helper…
1) Download helper (macOS):
One‑line install and run the local Helper (HTTPS, 127.0.0.1:5050).
/bin/bash -lc 'curl -fsSL https://lichanghengxjtu.github.io/SWEf/nonllmplatform/downloadhelper/install_helper.sh | bash'
2) Restart helper:
Kickstart the LaunchAgent to restart the Helper immediately.
launchctl kickstart -k gui/$(id -u)/com.sweperf.helper
3) Uninstall helper:
Stop the LaunchAgent and remove installed files and work dir.
launchctl bootout gui/$(id -u)/com.sweperf.helper || true; rm -f ~/Library/LaunchAgents/com.sweperf.helper.plist; rm -rf ~/.sweperf ~/SweperfWork
After installation, go to Non‑LLM Bench to run Docker checks.
